# Welcome to GoTrackIt !

GoTrackIt 是一个基于隐马尔可夫模型实现的地图匹配包，通过概率图建模的方式将车辆的GPS轨迹数据匹配到道路路网，获取车辆的标准化时空轨迹， 可以有效支撑出行导航、交通监测、交通治理、碳排核算、交通建模等方向.
